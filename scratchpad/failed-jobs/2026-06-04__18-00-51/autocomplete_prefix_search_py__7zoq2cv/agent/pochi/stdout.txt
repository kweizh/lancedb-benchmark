## What it does

```python
def autocomplete(prefix: str, k: int = 10) -> list[dict]
```

**Step 1 — SQL prefix match:**
- Connects to LanceDB using `LANCE_DB_URI`, opens `MOVIES_TABLE` and `PREFIX_TABLE` from env vars.
- Filters movies with `title_lower LIKE '{prefix_lower}%'` (case-insensitive via the pre-seeded `title_lower` column).
- Sorts all M matches by `popularity` descending.

**Step 2a — M ≥ k:** Returns top-k rows, each with `source="prefix"`.

**Step 2b — M < k (mixed mode):**
- Includes all M prefix rows (`source="prefix"`) ordered by popularity.
- Looks up `lower(prefix)` in `prefix_vectors` to get the precomputed 32-dim query vector.
- Runs a vector search (L2) on the movies table with a `prefilter` excluding already-returned ids, fetching exactly `(k − M)` more results with `source="semantic"`.
