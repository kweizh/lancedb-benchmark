## What was built

**`build_snapshots(db_path, table_name)`**
1. Connects to the LanceDB instance and drops any pre-existing table (idempotent).
2. Creates a shared `numpy.random.default_rng(seed=2026)` RNG.
3. **Phase 1 (v1_baseline)** — inserts 50 rows (`id` 0–49, `text="doc-{id}"`, 16-d float32 vectors) and tags the resulting version 1.
4. **Phase 2 (v2_extended)** — appends 20 rows (`id` 50–69) from the same RNG sequence and tags version 2.
5. **Phase 3 (v3_pruned)** — calls `table.delete("id < 5")` (removes 5 rows) and tags version 3.

**`diff(db_path, table_name, tag_a, tag_b)`**
- Opens the table, calls `table.checkout(tag_a)` / `table.checkout(tag_b)` to time-travel to each snapshot.
- Reads the `id` column from each, computes set differences, then calls `table.checkout_latest()` to restore the live view.
- Returns `{"added_ids": [...], "removed_ids": [...], "common_count": N}`.
