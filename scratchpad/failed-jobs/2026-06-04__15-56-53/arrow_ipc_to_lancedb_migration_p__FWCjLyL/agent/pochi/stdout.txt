## What the script does

1. **Reads** the Arrow IPC stream at `/app/source/dataset.arrows` using `pyarrow.ipc.open_stream()`.
2. **Creates** a LanceDB table named `events_${ZEALT_RUN_ID}` at `/home/user/myproject/lancedb_data` by passing the `RecordBatchStreamReader` directly to `db.create_table()` — no pandas detour, so `fixed_size_list<float32, 48>` is preserved exactly.
3. **Drops and recreates** the table on rerun so the command is idempotent.
4. **Verifies** schema parity field-by-field (name + Arrow logical type), setting `schema_match: true/false`.
5. **Loads** the query vector from `/app/query_vector.npy` as `float32[48]` and runs a top-5 L2 nearest-neighbour search via `table.search(...).metric("l2").limit(5)`.
6. **Prints** the JSON result to stdout.
