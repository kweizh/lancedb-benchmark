## Key design decisions
| Requirement | Implementation |
|---|---|
| Document embeddings | `client.embed(descriptions, model="voyage-3", input_type="document")` called once at catalogue-build time |
| Query embeddings | `client.embed([query], model="voyage-3", input_type="query")` called live at search time |
| Idempotent table build | `if TABLE_NAME in db.table_names(): return db.open_table(TABLE_NAME)` |
| Vector dimension | `pa.list_(pa.float32(), 1024)` — confirmed `fixed_size_list<float32>[1024]` in schema |
| Ranking | `table.search(query_vec).limit(k).to_list()` — native LanceDB cosine/L2 ANN search, rank-1 first |
