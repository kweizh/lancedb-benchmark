**Flow**
1. **Parse CLI args** — `--user-id`, `--query-vec` (`.npy` path), `--k`, `--output`.
2. **Read `BLEND_ALPHA`** from environment (default `0.3`).
3. **Load recent history** — queries the `user_history` table filtered by `user_id`, sorts by `ts` descending, takes up to 10 rows, returns the `item_id` list.
4. **Fetch embeddings** — looks up those item IDs in the `items` table via a `WHERE id IN (...)` pre-filter and stacks the 64-d vectors.
5. **Taste vector** — `mean` of the fetched embeddings, then blended:  
   `final = (1 - alpha) * query + alpha * taste`  
   If the user has no history, `final = query` (pure fallback).
6. **Vector search** — calls `items.search(final_vec).where("id NOT IN (...)", prefilter=True).limit(k)`, which excludes all previously seen items at the index level.
7. **Write output** — serialises the ranked integer IDs as a JSON array.
