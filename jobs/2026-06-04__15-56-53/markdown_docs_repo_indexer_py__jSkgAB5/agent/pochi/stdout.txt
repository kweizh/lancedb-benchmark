## `/home/user/myproject/indexer.py`

### What it does

| Responsibility | Details |
|---|---|
| **Walk & parse** | Recursively finds every `.md` under `/app/docs/`, extracts the `# Title` as `doc_title`, and splits on `## Section` headers into individual chunks. |
| **Embed** | Sends all chunks in a single batch to the real OpenAI `text-embedding-3-small` API (1 536-dim vectors). The embed text is `section_title + "\n\n" + content`. |
| **Persist** | Stores rows into a LanceDB table at `/home/user/myproject/lancedb/` named `docs_sections_${ZEALT_RUN_ID}` with schema: `repo_path`, `doc_title`, `section_title`, `content`, `embedding`. |
| **Idempotent** | On re-run the script checks whether the table already exists with the expected row count and skips re-indexing if so. |
| **`search(query, k)`** | Embeds the query, runs a cosine-similarity vector search via LanceDB, and returns a list of `k` dicts with `repo_path`, `doc_title`, `section_title`, and `score` (1 − cosine distance), ordered most-relevant first. |
