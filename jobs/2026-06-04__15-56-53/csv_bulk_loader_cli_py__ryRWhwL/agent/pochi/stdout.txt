## Implementation

### `ingest` subcommand
1. Reads the CSV with `csv.DictReader` — every row is preserved exactly once with no deduplication
2. Validates the `--text-col` exists in the CSV header
3. Calls the OpenAI `text-embedding-3-small` endpoint in configurable batches (default 100 rows/call), with exponential-backoff retry (up to 5 attempts) on transient failures
4. Assembles a typed PyArrow table: `id` (int64), `title`, `body`, `category`, `published` (utf8), `vector` (fixed-size list of 1536 float32 values)
5. Writes to `lancedb.connect('/home/user/loader_project/lance_db')` with `mode="overwrite"` so re-runs on the same table name are safe

### `search` subcommand
1. Embeds the `--query` string with the same OpenAI model (single call)
2. Opens the named LanceDB table and calls `tbl.search(query_vector).limit(k).to_list()`
3. Maps LanceDB's `_distance` field to `score`
4. Prints **only** the required JSON object to stdout; all progress/error messages go to stderr:
```json
{
  "query": "...",
  "k": 10,
  "results": [
    {"id": 42, "title": "...", "category": "...", "published": "...", "score": 0.312}
  ]
}
```
